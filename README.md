# 📝 Blog Website using Django

A clean, dynamic, and fully functional blog platform built with Django. This project showcases the power of Django in creating scalable web applications with features that make content publishing simple and engaging.

## 🚀 Overview

This blog website allows users to read, create, and manage posts with ease. It is designed with a focus on simplicity, readability, and performance. Whether you're sharing ideas, tutorials, or stories, this platform provides a solid foundation.

## ✨ Features

* 📰 Create, edit, and delete blog posts
* 📚 Organized post listings with detail views
* 🔐 User authentication (login/logout)
* 🖊️ Author-based content management
* 📅 Timestamped posts
* 🎨 Clean and responsive UI
* ⚙️ Admin panel for easy backend management

## 🛠️ Built With

* Python
* Django
* HTML, CSS
* SQLite (default database)

## ⚡ Getting Started

1. Clone the repository

   ```
   git clone https://github.com/your-username/Blog_website_using_Django.git
   ```

2. Navigate to the project directory

   ```
   cd Blog_website_using_Django
   ```

3. Install dependencies

   ```
   pip install -r requirements.txt
   ```

4. Run migrations

   ```
   python manage.py migrate
   ```

5. Start the development server

   ```
   python manage.py runserver
   ```

6. Open your browser and go to

   ```
   http://127.0.0.1:8000/
   ```
